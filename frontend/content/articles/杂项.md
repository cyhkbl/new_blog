## 通过权限更改删除文件

电脑上总会有一些垃圾文件需要TrustInstaller权限或者System权限才能删除，普通用户权限无法删除这些文件。如下图：

![无法删除的文件](/images/delete2.png)

用管理员权限打开PowerShell或命令提示符，执行以下命令：

```pwsh
# 获取文件夹所有权
takeown /f "路径" /r /d y

# 授予Administrators组完全控制权限
icacls "路径" /grant Administrators:F /t

# 继承权限设置
icacls "路径" /inheritancelevel:e /t
```

将上述命令中的“路径”替换为需要删除的文件或文件夹的实际路径。执行完这些命令后，您应该能够删除或修改这些文件。

## 跳过OOBE

第一种方式：输入 `start ms-cxh:localonly`并回车：

第二种方式：先输入 `reg add HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\OOBE /v BypassNRO /t REG_DWORD /d 1 /f`，然后输入 `shutdown /r /t 0` 并回车。（其实就是原有的 `BypassNRO.cmd` 后两行）

第三种方法（Windows 11 25H2前的版本可用）：输入`OOBE\BypassNRO`并回车。

## Github Copilot 提高敏感度（键入后每次都出现建议）

1. 打开 VSCode 设置（Ctrl + Shift + P），用户设置
2. 在 settings.json 中添加以下内容：

```json
"github.copilot.inlineSuggest.showInlineCompletions": "always",
"editor.suggestOnTriggerCharacters": true,
"editor.quickSuggestions": {
    "other": "on",
    "comments": "off",
    "strings": "on"
}
```
